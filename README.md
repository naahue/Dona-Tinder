# DonaTinder

1. `cd Donatinder`
2. `python3 -m pip install -r requirements.txt`
3. `python3 manage.py migrate`
4. `python3 manage.py createsuperuser` (optional: access `/admin/`)
5. `python3 manage.py runserver`
6. Open the app at [http://127.0.0.1:8000/](http://127.0.0.1:8000/) and the admin at [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)

On Windows, if `python3` is not on your PATH, use `py -3` instead of `python3` in steps 2–5.
