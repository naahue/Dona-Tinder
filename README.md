# DonaTinder

1. `cd Donatinder`
2. `py -3 -m pip install -r requirements.txt`
3. `py -3 manage.py migrate`
4. `py -3 manage.py createsuperuser` (opcional: acceso a `/admin/`)
5. `py -3 manage.py runserver`
6. Abrí la app en [http://127.0.0.1:8000/](http://127.0.0.1:8000/) y el admin en [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
